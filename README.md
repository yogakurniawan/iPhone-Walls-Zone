# React Airbnb Navbar

A Navbar built with React and mimicking Airbnb navbar style.

## How to use

Download the example [or clone the repo](https://github.com/yogakurniawan/react-airbnb-navbar):

Install it and run:

```bash
npm install
npm run start
```

or

```bash
yarn install
yarn start
```
